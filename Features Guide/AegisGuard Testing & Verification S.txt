🌐 Production Deployment Steps
To deploy this setup globally so that it's accessible anywhere:

Step 1: Copy files to your Production VM
Clone the codebase or upload the folder to your remote virtual machine (AWS, GCP, DigitalOcean, etc.).

Step 2: Set up Production Environment
Configure a .env file on the server with production secrets:

bash

JWT_SECRET=super-secret-random-jwt-signing-key
GEMINI_API_KEY=your_production_gemini_api_key
ENVIRONMENT=production
Step 3: Run the Docker Compose Stack
Execute the following command to build the AegisGuard container and start Postgres, Redis, Prometheus, and Grafana:

bash

docker compose up -d --build
Step 4: Bootstrap the Production Admin
Run the CLI within the app container to provision your admin dashboard credentials:

bash

docker compose exec mcp-server python -m secure_mcp_server.cli bootstrap --username admin --password securepassword123 --email admin@domain.com
Your REST API Control plane will be active at http://your-server-ip:8000, and you can plug this governed server into any client globally!
